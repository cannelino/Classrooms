# MXInkIntegration

Sample of how to use Logitech MXInk with Fusion.
See https://logitech.github.io/mxink/UnityIntegration.html for their latest integration package.

## Documentation

https://doc.photonengine.com/fusion/current/industries-samples/industries-addons/fusion-industries-addons-mxinkintegration
